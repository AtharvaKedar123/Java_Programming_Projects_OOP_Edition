# Practice Problem: Online Movie Ticket Booking System

## Problem Statement

CineBook is an online movie ticket booking platform that wants to automate the process of calculating the **final ticket price** for customers. The final amount depends on the **movie type**, **number of tickets booked**, **seat category**, and **booking type** (Normal or Premium).  
Write a Java program to implement the following design.

## Customer Class

### memberCustIdArr

- This is a static array (`String[]`) containing registered customer IDs.
- Initial values are as follows:

memberCustIdArr = {"CB101", "CB205", "CB309", "CB450", "CB520"}


**Note:**
- This array is supplied and hence no need to code
- Do not change the CASE of the elements

---

### memberDiscountArr

- This is a static array (`int[]`) containing discount percentage.
- Initial values are:

memberDiscountArr = {5, 10, 15, 20, 25}


- This array has one-to-one correspondence with `memberCustIdArr`

---

### calculateDiscount(totalAmount)

- Check whether `custId` is present in `memberCustIdArr`
- Identify the corresponding discount percentage
- Calculate discount on `totalAmount`
- Return discounted amount
- If customer ID is not found, return `-1`

## PremiumBooking Class

### calculateFinalAmount()

- Logic:
  1. Identify ticket price using `identifyTicketPrice()`
  2. If ticket price is `-1`, return `-1`
  3. Calculate baseAmount = ticketPrice × noOfTickets
  4. Add premiumCharge of **50 per booking**
  5. Apply **8% entertainment tax**
  6. Invoke `calculateDiscount()` of Customer class
  7. If discount returned is `-1`, return amount without discount
  8. Else return discounted amount

---
