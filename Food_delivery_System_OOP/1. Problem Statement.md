###### 1\. Problem Statement



Design and implement a Food Delivery Management System using object-oriented programming concepts.



* The food delivery platform offers different types of food orders to customers.
* The total bill amount for an order depends on:
* The food category
* The number of items ordered
* Optional express delivery service
* Optional festival discount



The system must calculate the final bill amount by strictly following the rules described in this problem.

###### 5\. Static Data (CustomerDetails Class)

**i. Valid Customer ID Prefixes**

* validPrefixArr = {"FD-C", "FD-P", "FD-X"}
* A customer ID is valid only if it starts with one of the above prefixes.



ii. Food Types

* foodTypeArr = {"Veg", "NonVeg", "Combo"}



iii. Base Price Per Item

* basePriceArr = {150, 250, 400}



**Each price corresponds to the food type at the same index**



###### **6. Implementation Details**



###### i. CustomerDetails Class()

**Method: validateCustomerId()**



* Returns true if customerId starts with any prefix in validPrefixArr
* Otherwise, returns false



###### ii. Order Class (Abstract)

**Method: getFoodIndex(String type)**



* Returns the index of the food type from foodTypeArr
* Comparison must be case-insensitive
* Returns -1 if the food type is invalid



**Method: calculateBillAmount**()



* Abstract method
* Must be implemented by all subclasses
* Returns the final bill amount as a double



###### iii. RegularOrder Class()

**Method: calculateBillAmount()**



Step-by-step logic:

* Validate the customer ID
* Retrieve the food index
* If validation fails or index is invalid, return -1
* Calculate base cost:
* BaseCost = basePrice × quantity





**If isExpressDelivery is true, add:**



* deliveryCharge
* Apply 5% service tax:
* FinalAmount = BaseCost × 1.05
* Return the final amount



###### iv. PremiumOrder Class

**Method: calculateBillAmount()**



**Step-by-step logic:**

* Validate the customer ID
* Retrieve the food index
* If validation fails or index is invalid, return -1
* Calculate base cost:
* BaseCost = basePrice × quantity
* Apply festival discount:
* DiscountedAmount = BaseCost − (BaseCost × festivalDiscount / 100)
* Add a fixed premium packaging charge of 50
* Return the final amount
